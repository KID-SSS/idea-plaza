# 创意广场 - Reddit精选产品创意

每日自动收集Reddit上的产品创意、需求、点子，并进行分析评分。

## 部署方法

### Vercel (推荐)
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/idea-plaza)

1. Fork 这个仓库
2. 点击上面的按钮
3. 完成！

### Cloudflare Pages
1. Fork 这个仓库
2. 进入 Cloudflare Dashboard → Pages
3. 创建项目 → 连接 Git
4. 选择这个仓库，直接部署

### GitHub Pages
1. Fork 这个仓库
2. Settings → Pages → Source 选择 main 分支
3. 保存，等待部署完成

### Netlify
1. 拖拽 `index.html` 到 [Netlify Drop](https://app.netlify.com/drop)
2. 完成！
